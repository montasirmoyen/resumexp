# What is this project?

This website uses AI to analyze a user's resume.
By uploading a PDF/DOCX/TXT file, the file is sent to a parsing module which extracts the text. That text is then sent to the AI pipeline where the AI reads the text along with a prompt, responding with a JSON object which is sent to the frontend to visualize.

# Current main features

- Functional auth
- Comprehensive AI-powered analysis of the resume
- Job match score if provides a job description

# Current limitations
- No data persistence
